# University of Cape Town - BSc Computer Science

A pragmatic, high-level summary of Computer Science courses at the University of Cape Town.

I took these courses while studying a BSc in Computer Science & Game Development from 2012 to 2014. The contents of this site come exclusively from my handwritten course notes.

# Philosophy

High Level

This document aims to serve as a high level overview of course contents.

Concise

External resources should be used for low level details.

Comprehensive

All of the major topics that formed part of the degree should be covered.

Consistent

Different types of content should be visualized consistently and distinctively.

# Style Guide

Core to this project is a consistent, efficient layout of content which provides semantics & context.

Most of a CS course content can be represented using a small set of constructs:

  • Hierarchy - content can be split into parent/child relationships.
  • Algorithms - psuedocode and mathematical notation.
  • Definitions - a lot of the content is the definition of a field/topic/subject.
  • Lists - a set of components that together form a higher level concept.

# Hierarchy

Hierarchy is achieved using groups, pages, titles & subtitles.

The hierarchy generally follows the following format:

  • Course (group)

    • Module (page)

      • Section (title)

        • Topic (subtitle)

# Algorithms

  • Algorithms are described using pseudocode.
  • Mathematical notation is represented using the equivalent programming convention.

Both are represented using code blocks:

// pseudocode for max in array
max = 0
for val in array:
    if val > max
        max = val
return max

// mathematical notation
powers: n^x
subscript: n(x)
